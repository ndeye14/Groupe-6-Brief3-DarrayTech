Presentation

Sans frameworks comme Angular, bibliothèques comme React ou connaissances expertes en JavaScript en général. Juste avec une fonctionnalité magique native JavaScript appelée « Composants Web » (ou « éléments HTML personnalisés »)

Les composants Web sont une combinaison de diverses spécifications intégrées au navigateur. Démarrer avec ces fonctionnalités est un jeu d'enfant et vous serez rapidement en mesure de créer vos propres éléments HTML personnalisés puissants et réutilisables (même dans plusieurs projets !).

Ce cours couvre tout !

De tels éléments personnalisés ne remplacent cependant pas Angular, React ou Vue - au lieu de cela, vous pouvez facilement les utiliser dans N'IMPORTE QUEL projet Web, y compris les projets utilisant de tels frameworks et bibliothèques.

De manière détaillée, ce cours couvre les structures suivantes :

  • comment créer des éléments HTML personnalisés légers et réutilisables avec des fonctionnalités de navigateur natives
  • comment créer des composants Web de toutes complexités - d'une simple info-bulle aux modaux ou tiroirs latéraux
  • comment transmettre des données dans vos propres composants Web et les utiliser là-bas
  • comment émettre vos propres événements personnalisés que vous pouvez écouter en JavaScript
  • comment utiliser le Shadow DOM pour étendre vos styles CSS à vos éléments personnalisés
  • comment utiliser Stencil.js pour obtenir un flux de travail de développement beaucoup plus simple
  • comment utiliser les nombreuses fonctionnalités fournies par Stencil.js pour créer des composants Web natifs de manière plus efficace
  • comment déployer/réutiliser vos propres composants Web dans N'IMPORTE QUEL projet en utilisant N'IMPORTE QUEL framework JavaScript comme Angular, React ou Vue (ou aucun !)

Vous apprendrez comment fonctionnent ces différentes structures de données, quand elles peuvent être
utiles, comment les construire et bien sûr nous explorerons également leur complexité temporelle (notation
Big O) !
