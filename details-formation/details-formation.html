Presentation

Lorsque vous travaillez avec JavaScript, vous travaillerez constamment avec des structures de données .

Cela inclut les éléments intégrés tels que les tableaux, les objets, les cartes ou les ensembles, mais - surtout si vous plongez plus profondément dans JavaScript - également les structures de données personnalisées telles que les listes liées, les arbres ou les graphiques .

Ce cours couvre tout !

Nous récapitulerons les structures de données intégrées de base pour comprendre comment elles fonctionnent et quand les utiliser, avant d'explorer d'importantes structures de données personnalisées.

De manière détaillée, ce cours couvre les structures suivantes :

  • Tableaux (intégrés)
  • Objets (intégrés)
  • Ensembles (intégrés)
  • Cartes (intégrées)
  • Listes liées (personnalisées)
  • Piles (personnalisées)
  • Files d'attente (personnalisées)
  • Tables de hachage (personnalisées)
  • Arbres de base (personnalisés)
  • Arbres de recherche binaires (BST, personnalisés)
  • Arbres AVL (personnalisés)
  • Files d'attente prioritaires (personnalisées)
  • Tas (personnalisés)
  • Graphiques (personnalisés)

Vous apprendrez comment fonctionnent ces différentes structures de données, quand elles peuvent être
utiles, comment les construire et bien sûr nous explorerons également leur complexité temporelle (notation
Big O) !
